# PM 思维练习场

> 用真实产品案例磨砺你的产品经理思维 · AI 智能点评 · 即时反馈

**在线体验：** [neon-llama-cb9fb5.netlify.app](https://neon-llama-cb9fb5.netlify.app)

---

## 产品简介

PM 思维练习场是一款面向产品经理及产品爱好者的主动学习工具。区别于被动阅读产品文章，本应用要求用户**主动分析**微信、抖音、支付宝、小红书等主流 App 的真实产品决策，写下完整的思考过程，再由 AI 给出逐条点评和参考答案。

**核心理念：** 产品思维不是看会的，是练会的。

---

## 功能特性

### 📚 精选题库（16 道）
涵盖国内主流 App 的真实产品决策案例，按 5 个维度分类：

| 分类 | 示例题目 |
|------|---------|
| 功能设计 | 微信「拍一拍」的产品价值是什么？ |
| 增长策略 | 抖音推荐算法如何平衡创作者与消费者利益？ |
| 用户体验 | 小红书「瀑布流」双列布局的产品决策逻辑 |
| 商业模式 | 芝麻信用如何构建支付宝的护城河？ |
| 竞品分析 | 美团如何赢得外卖市场：对饿了么的策略复盘 |

- **3 种难度**：入门 / 进阶 / 高级，循序渐进
- **预估时长**：每道题 15-35 分钟，合理规划练习时间
- **支持搜索**：按题目、App 名称或标签快速筛选
- **随机练习**：解决选择困难，优先推荐未完成题目

### 🧠 结构化答题体验
- **背景介绍** + **练习题目** 清晰呈现，还原真实产品分析场景
- **可折叠思考框架提示**：每题配 4 条专业分析维度，默认收起不干扰思路
- **实时计时**：培养时间管理意识
- **字数进度条**：引导输出足够深度的分析（建议 300 字以上）
- **草稿自动保存**：每 800ms 写入 localStorage，刷新或意外关闭后内容完整恢复

### 🤖 AI 批改反馈
提交后进入批改页面，展示：
- **评分环**（0-100 分）+ 分析深度 / 框架完整 / 商业洞察三维子分
- **AI 点评**：逐条列出亮点（✅）、不足（❌）、进阶思考（💡）
- **参考框架**：优秀分析应覆盖的思考维度
- **参考答案要点**：完整的产品经理视角解读

### 📊 学习进度仪表盘
首页实时展示个人学习全貌：
- **完成进度环**：已完成 X/16 题，进度百分比
- **核心指标**：平均分（带等级色标）、最高分、连续练习天数（🔥）
- **总练习时长**：累计投入时间，换算为「等效课时」
- **近期得分趋势**：最近 7 次的折线走势图
- **分类覆盖条**：每个类别的完成比例 + 该类别平均分，直观暴露薄弱领域

### 💾 数据持久化
- 练习历史、草稿均存储于 localStorage，跨会话保留
- 题库卡片显示橙色圆点标记「有未完成草稿」
- 历史记录页按时间倒序展示所有已完成练习
